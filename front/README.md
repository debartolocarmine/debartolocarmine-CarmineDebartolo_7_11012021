## Setup

Pour personnaliser le port de votre application Reactjs, veuillez copier, à la racine du dossier **frontend**, le fichier **.env.default** et le renommer **.env**, puis, modifier à votre convenance la variable `PORT`.

## install

run `npm install`

## Start

run `npm start`
